# Simple chat

It is a client for a very simple chat.

You can find a server [here](https://github.com/HappyMary16/simple-chat-server).

## Requirements
- [Node JS](https://nodejs.org/uk/)

## Running the application

Download and install packages:
```npm install```

Run application:
```npm start```
